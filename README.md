## Run Locally

Prerequsities: https://docs.expo.dev/get-started/set-up-your-environment/

Clone the project

```bash
  git clone https://github.com/monarcode/miltiverse-app malltiverse
```

Go to the project directory

```bash
  cd malltiverse
```

Install dependencies

```bash
  yarn
```

Start Up Expo

```bash
  yarn start
```
